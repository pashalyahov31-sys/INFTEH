## Описание программы
Expense Tracker - это GUI-приложение для отслеживания личных расходов. Приложение позволяет добавлять, редактировать и удалять расходы, фильтровать их по категориям и датам, а также подсчитывать общую сумму расходов за выбранный период.

## Функциональность
- ✓ Добавление расходов (сумма, категория, дата)
- ✓ Редактирование и удаление записей
- ✓ Фильтрация по категориям
- ✓ Фильтрация по диапазону дат
- ✓ Подсчет суммы расходов за выбранный период
- ✓ Валидация ввода (положительная сумма, корректная дата)
- ✓ Сохранение и загрузка данных в формате JSON
